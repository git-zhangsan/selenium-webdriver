+  Key.chord( ...var_args )

>  描述：模拟在“和弦”中同时按下多个按键。采取一系列的键或字符串，将每个值附加到一个字符串，添加和弦终止键（Key.NULL）并返回结果字符串

>  参数：var_args （string） 连接的关键序列
